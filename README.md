
Erstelle mir eine komplette interaktive One-Page-Webseite (HTML + CSS + JavaScript) für ein Schulprojekt mit dem Thema:

„Heißer Planet – schrumpfende Rechte (Klimawandel und Menschenrechte)“

Die Webseite soll modern, verständlich für Jugendliche und direkt für GitHub Pages geeignet sein (nur HTML, CSS, JavaScript – keine Frameworks).

⸻

1. Startseite & Login
	•	Willkommens-Startseite mit Animation
	•	Eingabefeld für Benutzername (Login)
	•	Begrüßung mit Namen („Willkommen, …“)
	•	Button zum Starten der App

⸻

2. Farbschema / Theme
	•	Auswahl zwischen Farb-Themen:
	•	Blau
	•	Grün
	•	Rot
	•	Auswahl wird gespeichert (localStorage)

⸻

3. Dark Mode
	•	Button zum Umschalten zwischen:
	•	Hellmodus
	•	Dark Mode

⸻

4. Inhalte (Lernbereiche)

Navigation mit Buttons (keine extra Seiten, alles auf einer Seite, Smooth Scroll):
	•	Klimawandel & Menschenrechte
	•	Recht auf Wasser
	•	Recht auf Wohnen
	•	Recht auf Gesundheit
	•	Was kannst du tun?

Inhalte kurz, einfach und verständlich für Schüler formuliert.

⸻

5. Mini-Spiel statt Quiz
	•	Spieltyp: „Richtig oder Falsch“
	•	Mehrere Fragen zum Thema Klimawandel & Menschenrechte
	•	Buttons: ✔ Richtig / ❌ Falsch
	•	Anzeige:
	•	aktuelle Frage
	•	Punktestand

⸻

6. Schwierigkeitsstufen (Level)
	•	Auswahl:
	•	leicht
	•	mittel
	•	schwer
	•	Jede Stufe hat eigene Fragen

⸻

7. Timer
	•	Jede Frage hat einen Countdown (z.B. 10 Sekunden)
	•	Wenn Zeit abläuft → automatisch nächste Frage

⸻

8. Sounds
	•	Sound bei:
	•	richtiger Antwort
	•	falscher Antwort

⸻

9. Mehrere Spielrunden
	•	Button „Neue Runde starten“
	•	Spiel kann beliebig oft neu gespielt werden

⸻

10. Score & Highscore
	•	Punkte werden gezählt
	•	Speicherung mit localStorage
	•	Highscore-Liste mit:
	•	Name
	•	Punktzahl
	•	Sortiert nach höchstem Score

⸻

11. Fortschritt & Animationen
	•	Animationen beim Laden (FadeIn / SlideUp)
	•	Fortschrittsanzeige (optional Balken)

⸻

12. Umfrage
	•	Frage z.B.:
„Würdest du dein Verhalten fürs Klima ändern?“
	•	Buttons: Ja / Vielleicht / Nein
	•	Anzeige der gewählten Antwort

⸻

13. QR-Code
	•	Automatische Generierung eines QR-Codes zur Webseite
	•	QR-Code basiert auf aktueller URL (window.location.href)

⸻

14. Design
	•	große Überschriften
	•	Buttons als Boxen mit Icons (Emojis)
	•	modernes, freundliches Design
	•	für Schüler verständlich

⸻

15. Technik

Gib mir den kompletten Code für:
	•	index.html
	•	style.css
	•	script.js

Anforderungen:
	•	Nur HTML, CSS, JavaScript
	•	Keine Frameworks
	•	Alles auf Deutsch
	•	Kommentare im Code zur Erklärung
	•	Smooth Scroll bei Navigation
	•	Responsive (Handy & PC)
	•	GitHub Pages kompatibel
